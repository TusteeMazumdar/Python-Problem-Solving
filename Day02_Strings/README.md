# Day 02 – Strings

## Problems Solved
1. Replace all 0's with 5 in a number
2. Change the string based on the first character
3. Concatenate two strings

## Concepts Practiced
- String traversal
- Type conversion (int ↔ string)
- Conditional logic
- Built-in string methods

## Learning Method
Problems were practiced on GeeksforGeeks and implemented independently
to improve string manipulation and logic building.
